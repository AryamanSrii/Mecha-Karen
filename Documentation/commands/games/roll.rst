Roll
=======
Karen scrambles up a random word depending on the chosen difficulty and you have to try guess it within 30s
 
Parameters
----------
**Member [Optional]**
 
**Layout:**
 ::
     -Roll [member]
 
Permissions
-----------
 **Bot**
 ::
     Embed Links
 
 **Author**
 ::
     Send Messages
 
Cooldown
--------
10 seconds.
