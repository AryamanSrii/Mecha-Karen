Fight
=======
Fight any member in the server using **Mecha Karen**!
 
Parameters
----------
Member [Required]
 
**Layout:**
 ::
     -Fight [member]
 
Permissions
-----------
 **Bot**
 ::
     Embed Links
     Add Reactions
 
 **Author**
 ::
     Send Messages
     Add Reactions
 
Cooldown
--------
20 seconds.

No aliases.
